# Microlearning Platform API

Итоговое домашнее задание по дисциплине «Разработка веб-сервисов и приложений» (НИУ ВШЭ).
Бэкенд-часть образовательной платформы, предоставляющая REST API для управления учетными записями, студентами и обработки обращений пользователей.

## Описание проекта

Проект представляет собой асинхронное веб-приложение, реализующее ролевую модель доступа (RBAC), безопасное хранение данных и фоновую обработку ресурсоемких задач. 
Реализован полный цикл CRUD-операций для работы с данными студентов, а также эндпоинты для сбора обращений технической поддержки.

## Стек технологий

* **Язык:** Python 3.10
* **Фреймворк:** FastAPI
* **База данных:** SQLite (SQLAlchemy ORM)
* **Кеширование и фоновые задачи:** Redis, fastapi-cache2
* **Валидация данных:** Pydantic
* **Безопасность:** PyJWT, passlib (bcrypt)
* **Контейнеризация:** Docker, Docker Compose
* **Тестирование:** Pytest, TestClient

## Структура проекта

Проект декомпозирован согласно стандартам FastAPI:
* `app/routers/` — маршруты приложения (auth, students, appeals)
* `app/schemas/` — Pydantic-модели для валидации входящих/исходящих данных
* `app/database/` — модели SQLAlchemy, CRUD-операции и настройка сессий
* `app/core/` — настройки окружения, зависимости (RBAC) и безопасность
* `tests/` — функциональные тесты API

## Инструкция по запуску (Docker)

Проект полностью настроен для запуска в изолированной среде с помощью Docker Compose.

1. Убедитесь, что у вас установлены Docker и Docker Compose.
2. Клонируйте репозиторий и перейдите в корневую папку проекта.
3. Запустите сборку и старт контейнеров командой:

```bash
docker-compose up --build

```

4. После успешного запуска приложение будет доступно по адресу:
* **Swagger UI (документация):** http://localhost:8000/docs
* **ReDoc:** http://localhost:8000/redoc



## Запуск автотестов

Приложение покрыто функциональными тестами, проверяющими валидацию, бизнес-логику и корректность работы с базой данных (используется отдельная тестовая БД).

Для запуска тестов (убедитесь, что виртуальное окружение активировано и зависимости установлены):

```bash
pytest -v tests/test_api.py

```

## Дополнительный скрипт

В корне проекта находится скрипт `make_admin.py`, который позволяет быстро создать пользователя с правами `admin` для тестирования функционала.
